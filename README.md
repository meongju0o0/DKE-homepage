## Build, Run
```bash
sudo apt-get update
curl -sL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install nodejs
git clone https://github.com/meongju0o0/DKE-homepage
npm run build
serve -s build
```
